Beyond the Basics

Please press on the Picture button to see the picture of an eagle that I made using triangles.

Please press the Draw Kaleidoscope button and draw any pattern on the canvas to see the drawing made using triangles.

I made this picture first on a graph paper (shown below) and then using the code that I have written for this assignment to render triangles.

When the user presses the Draw Kaleidoscope button and subsequenlty draws on the canvas, the pattern that is drawn turns nto a beautiful kaleidoscopic pattern. I have achieved this effect by reflecting each point of the pattern across multiple lines passing through origin. The reflected points are drawn with different colors that are derived from the color selected by the user for the original pattern. This creates a multi-colored, symmetric kaleidoscopic-type image on the canvas. Moreover, the user can change the size of the point and the color to be used while drawing the pattern. This enables the user to have a fine-level of control over the kaleidoscopic image that they are creating.
